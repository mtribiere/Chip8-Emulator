# Chip8-Emulator
A simple chip8 emulator for entertainement and learning purposes writen in c++ using the SDL2 library.
Some rom have been included to play around with the emulator.

# # Compile from source
```
user@debian:~$ make
```

# # Play 
```console
user@debian:~$ ./chip8 <path_to_rom>
```
